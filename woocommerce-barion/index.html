Barion Payment Gateway for WooCommerce

This WordPress plugin allows your customers to pay via Barion Smart Gateway in your WooCommerce online store.

Get the plugin I need help Source code

Features

Barion payment

Adds Barion as a payment option to the WooCommerce checkout page.

Redirect

Redirects the user to the Barion payment page after checkout.

Barion callback

Handles the callback from Barion after payment, and sets the order status to “processing”, “completed” or “failed” respectively.

Refund

Refund payments via Barion.
